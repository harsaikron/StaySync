Portal Pages — Section 2 of 3

What each page does and what the patient/caregiver sees

🏠 /dashboard Home
  • Live camera feed grid (all registered cameras)
  • Latest Gemma 4 analysis per camera
  • Active alert banner (fall / wandering detected)
  • 🔊 "Speak now" button — plays guidance via TTS
  • Auto-speak toggle (hands-free mode)
📷 /cameras/setup Key Feature
  • Plug in ESP32-CAM via USB
  • "Connect Camera" button → Web Serial API
  • Enter home WiFi SSID + password
  • One-click flash credentials to device
  • Name the camera (e.g. "Living Room")
  • Camera appears in dashboard instantly
📋 /cameras
  • List all registered cameras
  • Online / offline status indicator
  • Last seen timestamp
  • Add new / remove camera
  • Link to individual camera live view
📝 /alerts
  • Full history of Gemma 4 detections
  • Severity badge: Info / Warning / Critical
  • Thumbnail of the triggering photo
  • Guidance text that was spoken
  • Timestamp + camera name
📱 /patient Patient-facing — simplified
  • Large text display of current guidance ("Time to take your medicine 💊")
  • Auto-speaks via Web Speech API when new guidance arrives (SSE)
  • Big "Repeat" button to replay last message
  • Calming background, no distracting UI
  • Works as a bookmark on the home screen — full-screen PWA
